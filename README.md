# Personal Portfolio - Ashish Makhija

A premium, full-stack developer portfolio built with React, Vite, and Framer Motion. 

## Features
- **Premium Dark Mode**: Elegant charcoal and indigo theme.
- **Dynamic Animations**: Smooth scroll reveals and micro-interactions.
- **Glassmorphism UI**: Modern floating design elements.
- **Responsive Design**: Optimized for all devices.
- **Featured Projects**: High-quality previews of ThreadFlow, Cloth Management, and Airport Automation.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) 

### Installation
1. Install dependencies (if not already done):
   ```bash
   npm install
   ```

## Development Commands

### Start Development Server
Run the project locally with hot-reloading:
```bash
npm run dev
```
The site will be available at `http://localhost:5173`.

### Production Build
Generate optimized production-ready files in the `dist` folder:
```bash
npm run build
```

### Preview Production Build
Locally preview the production build:
```bash
npm run preview
```

### Linting
Check code quality:
```bash
npm run lint
```

## Technologies Used
- **Vite**: Frontend build tool.
- **React**: Component-based UI.
- **Framer Motion**: Premium animations.
- **Lucide React (v0.470.0)**: Modern icon set.
- **Vanilla CSS (Modules)**: Custom design system.
